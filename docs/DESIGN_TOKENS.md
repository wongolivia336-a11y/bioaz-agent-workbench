# Design Tokens

Phase one preserves the imported DMPK visual baseline. The source of truth remains `app/globals.css` and `app/iteration.css`.

Rules:

- Use current color, border, radius, shadow, spacing, and typography values.
- Use Lucide React icons at the surrounding component's existing size and stroke width.
- Reuse established CSS classes before adding selectors.
- Keep task and file surfaces compact; avoid marketing cards and explanatory copy.
- Every interactive control needs hover, active, focus, disabled, and loading behavior where applicable.
- Responsive changes should preserve readable content width and fixed-format controls.
